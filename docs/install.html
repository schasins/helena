Install Helena

  1. Create a new Chrome profile for running your Helena programs. (Not a new Google account or a new GMail account! A new profile for your Chrome browser. See directions here.)
  2. Switch into the new Chrome profile, then install the extension from the Chrome Web Store. (Press the "Add to Chrome" button.)
  3. You're all set! Helena is installed, and you can click the new icon at the upper right to open the Helena control panel.

If you're a developer and you might want to modify how Helena works, you may prefer to follow the installation instructions for developers.

Once you've installed the Chrome extension, consider checking out the guide to using demonstrations to write Helena programs.
